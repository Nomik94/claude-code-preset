# careful gotchas — Claude가 자주 틀리는 패턴

### 1. 파이프 안의 위험 명령 미감지
❌ `echo "data" | xargs rm -rf`처럼 파이프 뒤에 숨은 위험 명령을 놓침
→ ✅ 파이프라인 전체를 검사해야 함

### 2. 변수 확장 후 위험 명령
❌ `$CMD` 변수에 `rm -rf`가 들어있는 경우 감지 실패
→ ✅ 변수 확장 결과도 검사 대상

### 3. 프로덕션 환경에서 careful 미활성화
❌ 프로덕션 서버 작업 시 `/careful` 호출 잊음
→ ✅ SSH/프로덕션 키워드 감지 시 자동 제안
